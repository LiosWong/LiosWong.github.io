Dubbo中telnet命令使用

Posted on 2019-10-24 | In Dubbo | Views:

从 2.0.5 版本开始，dubbo 开始支持通过 telnet 命令来进行服务治理。

命令列表

使用

  • 首先启动Dubbo提供者服务
  • telnet localhost 20880
  • 输入help可以查看命令列表
  • 使用invoke命令调用服务

由上可知,服务调用成功.

命令说明

  • ls

  • ps

  • cd

  • pwd

  • trace

  • count

  • invoke

  • select

  • status

  • log

  • help

  • clear

  • exit

  • shutdown
